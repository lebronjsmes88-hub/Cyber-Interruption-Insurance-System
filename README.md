# Cyber Interruption Insurance System

## Overview

The Cyber Interruption Insurance System is a decentralized parametric insurance platform built on the Stacks blockchain using Clarity smart contracts. This system provides automated coverage for business interruption caused by cyber incidents, including DDoS attacks, ransomware, and system outages.

## System Architecture

### Core Components

The system consists of three main smart contracts:

1. **cyber-incident-detector**: Integration with security monitoring tools and threat intelligence feeds
2. **downtime-measurement-oracle**: Automated system availability monitoring and downtime quantification  
3. **business-impact-calculator**: Revenue loss estimation and instant claim processing for cyber incidents
